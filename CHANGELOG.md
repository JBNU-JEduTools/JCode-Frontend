## 릴리스 노트 가이드

### 버전 번호

- **Major** (X.0.0): 호환성을 깨뜨리는 변경사항
- **Minor** (0.X.0): 하위 호환성을 유지하는 새로운 기능
- **Patch** (0.0.X): 하위 호환성을 유지하는 버그 수정

### 변경사항 유형

- **Added**: 새로운 기능
- **Changed**: 기존 기능의 변경
- **Deprecated**: 곧 제거될 기능
- **Removed**: 제거된 기능
- **Fixed**: 버그 수정
- **Security**: 보안 관련 수정
